# 数据结构和算法



### 1 前缀和

```
作用：
	可以快速求出一维数组中某一段和或者二维数组中子矩阵和
```



### 2 差分

```
概念：
	对于数组 a1, a2, a3, .., 构造数组 b1, b2, b3, ..
	使得 ai = b1 + b2 + b3 + ... + bi
	
	note: b数组为a的差分，a数组为b数组的前缀和

构造:
	b1 = a1
	b2 = a2 - a1
	b3 = a3 - a2
	
作用:
	如果需要对 a 数组中某一段 l-r 进行统一的 +c 操作
	现在只需要做 b[l] + c, b[r+1] - c, 两个操作即可
```
